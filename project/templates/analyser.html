Assumptions and Limitations:

  • The length of the password directly determine the maximum score the password can achieve.
  • ASCII's 32 symbols (not including space character) are used to determines which characters are counted as special characters in entropy check and composition check.
  • For the entropy calculation, the charset range is determined from the password's inclusion of the following 4 character type pools: 10 numerics, 26 lower, 26 upper, and 32 symbols.
  • Password de-substitution is limited to 200 variants to reduce loading times.
  • Blocklist check uses Have I Been Pwned's password API with k-anonymity to check if the submitted password (and its de-substituted variants) is commonly used.
  • Pattern checks are limited to three specific pattern types: sequential chars, keyboard patterns, and consecutive repeated chars (3+).
